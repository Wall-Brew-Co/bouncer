## 1.1.1 - 2024-10-13

* Fixed

## 1.1.0 - 2024-10-13

* Added
  * `fix` command to [fix issues with existing projects.](https://github.com/Wall-Brew-Co/bouncer#fix-a-project-with-bouncer)

  * Support for loading configuration from `.wallbrew/bouncer/config.edn`.
